# Phase 4 Round 1: R3 做空实验 — 失败记录

## 改动
- 3S 恢复开空能力（中枢向下离开后回抽不破ZD → 开空）
- 2S 保持仅平多
- 开空额外过滤: 15m diff < 0

## 结果

### 版本1: 3S+2S 全部开空
- Total: 2299 pts (vs 9902 baseline) ← 灾难
- p2209: -241 (从+7110 暴跌)
- p2301: -430 (微改善)
- 交易数翻倍

### 版本2: 仅3S 开空, 2S 只平多
- Total: 1030 pts ← 更差
- p2209: -1014
- p2301: -869

## 失败原因分析
1. **p2209 在极端下跌中靠做多反弹赚7110pts** — 开空破坏了反弹做多的窗口
2. 15m diff < 0 过滤不够严格 — 极端行情中diff震荡剧烈
3. 做空的止损更难控制 — 反弹时空头被止损后无法及时转多

## 结论
**做空在当前数据集上不可行**。p2209 的极端贡献来自"在下跌中抓反弹"，
这与做空逻辑天然矛盾。放弃做空方向，回到"降频+过滤"路线。
