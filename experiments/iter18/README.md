## 迭代背景

### iter17 核心结论
1. **S27 跳空 ATR 自适应失败** (TOTAL -110.9 pts)
   - "放宽止损"思路不可行
   - 放宽止损 = 单笔亏损更大 > 减少被震出的收益
   
2. **节后信号胜率不低（66.7%）**
   - 问题不在信号质量，而在止损被穿透

3. **短持仓(<5 bars)全亏**
   - 30笔全亏，胜率 0%
   - 是主要亏损来源之一
