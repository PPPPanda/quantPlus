# iter9 Phase 1: 结构审计

## 审计方向：节假日保护 + 止盈策略 + 参数优化

### 1. 节假日保护分析

**当前状态**: 策略无任何节假日保护机制。

**诊断数据** (7合约):
- 总 overnight PnL = 68,680 (6,868 pts)，占总利润 11,117 pts 的 **62%**
- Overnight trades 总体是巨大利润来源

**关键 overnight 亏损案例**:
- p2501: 2024-10-11(周五夜) → 2024-10-14(周一): pnl=-1360（十一长假后首周）
- p2209: 2022-06-01(夜) → 2022-06-02: pnl=-1100（端午节前后）  
- p2209: 2022-07-01(周五夜) → 2022-07-04(周一): pnl=-920
- p2501: 2024-06-25(夜) → 2024-06-26: pnl=-800

**结论**: 节假日保护弊大于利。overnight持仓是主要利润来源，如果节假日前平仓，会丢掉大量利润。例如 p2505 的最大赢家 3180 pts 就是春节前开仓、节后平仓。

### 2. 止盈策略分析

**当前出场机制**:
- P1 硬止损（回踩点 - buffer）
- ATR trailing stop（activate=2.5×ATR, trail=3.0×ATR）
- 3S/2S 缠论信号平仓
- 笔底结构trailing（use_bi_trailing=True）

**问题**: 当前策略擅长抓大趋势，大赢家动辄2000-3800 pts。如果加入固定止盈（如10×ATR），会砍掉这些大赢家的尾巴。

**止盈机会评估**:
- 大部分最佳交易持仓时间 > 500分钟（> 100根5m bar）
- trailing stop 已在趋势末端发挥止盈效果
- 加入主动止盈可能导致频繁出场→重入场成本高

**结论**: 当前 trailing stop 已经是一种自适应止盈。不建议加入固定目标止盈。
