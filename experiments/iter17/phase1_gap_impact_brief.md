# iter17 Phase 1 多方讨论议题：节假日跳空对缠论结构的影响

## 背景数据（13 合约统计）

| 指标 | 节假日跳空 | 普通跳空 | 差异 |
|------|-----------|----------|------|
| 样本数 | 23 | 481 | - |
| 平均幅度 | 134.2 pts | 41.6 pts | **3.23x** |
| 当天回补率 | 34.8% | 71.7% | **节假日难回补** |
| 占当天波动比 | 0.44 | 0.21 | **节假日跳空占比更大** |

## 核心问题

节假日跳空可能对缠论结构判断造成以下影响：

### 1. 分型/笔层面
- 跳空后第一根 K 线幅度异常大（占当天波动 44%）
- 可能形成"假分型"（极端高点/低点，但非趋势拐点）
- 笔的端点可能被错误标定

### 2. 中枢层面
- 大跳空可能让价格直接"穿越"中枢区间
- 中枢的上下轨计算可能被扭曲
- 可能产生"假突破"或"假跌破"信号

### 3. 背驰判断层面
- MACD 在跳空后会产生异常读数（突然放大/缩小）
- 面积背驰的计算可能因跳空而失真
- 可能产生错误的背驰信号

### 4. 买卖点层面
- 基于错误的笔/段结构产生错误的买卖点
- 节后开盘的信号可能是"假信号"

## 讨论要点

1. **影响程度评估**：这个问题对当前策略收益的影响有多大？是主要矛盾还是次要矛盾？

2. **现有策略是否已有隐性保护**：
   - 连亏断路器（CB）是否已经间接保护了部分节后错误信号？
   - ATR 止损是否已经适应了部分波动？

3. **改进方案优先级**：
   - S25 节后缓冲期（节后前 N 根 5m bar 不开新仓）
   - S26 跳空检测（检测到大跳空后，忽略前 N 根 bar 的分型信号）
   - S27 ATR 自适应（跳空后临时放大 ATR 基准）
   - S28 结构重置（节后强制重新构建笔/段/中枢）

4. **实施风险**：
   - 节后观望可能错过大行情（iter9 结论：Overnight 贡献 62% 利润）
   - 过度保护可能导致收益下降
